# Server 101
## We use the `express` library to build servers 

### Steps to build a siimple server 
Follow this steps in the project root folder (demo)
- `npm init -y`
- `npm i express`
- `npm i -D nodemon`
- `touch server.js`
- add the code to build a server (see **demo/**)
- `npx nodemon server.js`
